# 项目依赖说明：
# - Flask：提供 Web 服务接口和简单的前端页面。
# - Jinja2：HTML 模板渲染引擎，与 Flask 配合使用。
# - jieba：中文分词与词性标注，用于 POS 规则。
# - pypinyin：提供汉字转拼音能力，可作为扩展使用。
# - PyYAML：加载自定义规则的 YAML 文件。
# - openpyxl：将检查结果导出为 Excel（XLSX）格式。
# - pycorrector：集成 MacBERT4CSC 中文拼写纠错模型。
#   注意：首次加载模型需要联网下载权重，之后可离线使用。
# - transformers：支持直接加载 HuggingFace 权重，如需要可切换。
# - torch：Transformer 模型的运行时依赖。
